DLipid

Dynamics of the ethanolamine glycerophospholipid remodeling network

Welcome to DLipid Pages.

Here we could introduce a summary. something like:

Acyl chain remodeling in lipids is a critical biochemical process that plays a central role in disease. However, remodeling remains poorly understood, despite massive increases in lipidomic data. With this application, the novel dynamic network of ethanolamine glycerophospholipid (PE) remodeling is determined. The model uses a set of ordinary differential equations based on the assumptions that (1) sn1 and sn2 acyl positions are independently remodeled; (2) remodeling reaction rates are constant over time; and (3) acyl donor concentrations are constant. The avaliable tool uses a novel fast and accurate two-step algorithm to automatically infer model parameters. This is the first such method applicable to dynamic phospholipid lipidomic data.

How to use it?

$ cd DLipid/bin
$ java -jar DLipid.jar inputFile minimumTH [maximumTH]

A) inputFile:

Path. If only the file name is written, default input folder will be selected: ../DATA/inputFile Specificiation

                      time1           time2           . . .

      specie1         mean    std     mean    std

      specie2         mean    std     mean    std

       . . .

B) Thresold used by FluxNetworkApplciaton: goes from the minimum thresold to the maximum one with the step size 0.1.

  • maximumTH is optional; if it is not indicated, only the minimum threshold will be computed.

Example: 

$ java -jar DLipid.jar 14_0-14_ 0.txt 0.2

Interpreting the results:

$ cd "DestinationFolder"/RESULTS/"inputFile"

A) "DestinationFolder depends on the inputFile path. Using 14_0-14_0.txt as example

  • if it is inside of DATA folder, ./DLipid/Results/14_0-14_0/
  • ohterwise; "inputFilePath"/14_0-14_0/

B) Output files:

  • *.txt: LULU please describes
  • *.eps: LULU please describes
  • *.graphml: this file is an xml file format which stored flux network. See Graphml File Extension for more detail.
  • *.sif: codification of the remodeling network. The meaning of each line is: source1 parameter1 target1
